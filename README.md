# 🌴 VacayFinder

VacayFinder is an AI-powered vacation planning tool that helps you plan your perfect holiday! 🚀  
It uses Google's **Gemini API** and Python to suggest personalized travel destinations, itineraries, and activities based on your preferences.

---

## 📌 Features

- 🌍 AI-based vacation suggestions
- 🗺️ Customizable trip plans based on user input
- 💬 Chat-like interface for destination queries
- 📊 Real-time recommendation updates via Gemini API

---

## 🧰 Tech Stack

- **Python**
- **Gemini API (Google AI)**
- **Flask (optional, if you're using it for a web interface)**
- **HTML/CSS/JavaScript** (if applicable for UI)

---

## 🚀 How to Run the Project

### 1️⃣ Clone the Repository:
```bash
git clone https://github.com/your-username/VacayFinder.git
````

### 2️⃣ Install Dependencies:

```bash
pip install -r requirements.txt
```

### 3️⃣ Set up your Gemini API key:

* Get your API key from [Google AI Studio](https://makersuite.google.com/app/apikey)
* Create a `.env` file and add:

  ```
  GEMINI_API_KEY=your-api-key-here
  ```

### 4️⃣ Run the Program:

```bash
python app.py
```
